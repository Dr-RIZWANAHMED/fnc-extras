Now we imagine that is unknown and that the blurred is given. We want to invert the blur transformation using the transformation itself. But we have to translate between vectors and images each time.
vec = @(X) reshape(X,m*n,1);

Now we apply gmres to the composite blurring transformation T.
y = gmres(T,vec(Z),50,1e-5);
